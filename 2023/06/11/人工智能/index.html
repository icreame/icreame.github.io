人工智能复习指南

第一章

<重点>智能的本质（特性）：

  1. 智能是具有感知能力的。
  2. 智能具有记忆和思维能力。
  3. 智能具有学习能力、自适应能力及行为能力。

人工智能的研究目标：

人工智能研究的远期目标与近期目标是相辅相成的。

​ 远期目标为近期目标指明了方向，而近期目标的研究为远期目标的最终实现奠定了基础，作好了理论及技术上的准备。

​ 近期目标的研究成果不仅可以造福于当代社会，还可以进一步增强人们对实现远期目标的信心，消除疑虑。

人工智能的定义：

1．从学科角度看，人工智能是研究如何制造出人造的智能机器或智能系统来模拟人类智能活动的能力，以延申人类智能的学科。

2．从能力角度看，人工智能是智能机器所执行的的通常与人类智能有关的智能行为，如判断、推理、证明、识别、感知、理解、通信、设计、思考、规划、学习和问题求解等思维活动。

<重点>人工智能发展史：

（1）孕育时期（1956年以前）：

  • 亚里斯多德提出了形式逻辑的一些主要定律，其中最著名的是三段论。

    ​ 三段论是以真言判断为其前提的一种演绎推理，它借助于一个共同项，把两个直言判断联系起来，从而得出结论。

    ​ 例如：一切金属都是能够熔解的；铁是金属；所以，铁是能够熔解的。

  • 英国科学家培根提出了归纳法。

  • 英国科学家布尔创立了布尔代数。

  • 英国科学家图灵，主要贡献图灵测试和图灵机。

  • 美国科学家莫客力、埃柯利研制了第一台电子计算机ENIAC。

（2）形成时期（1956-1969）：

  • 1956年，美国的达特茅斯大学召开第一次人工智能研讨会，标志着人工智能学科的诞生。
  • 1960年，美国的麦卡锡发明了人工智能语言LISP。
  • 1960年，纽厄尔编制了通用问题求解程序GPS，可求解11种不同类型的问题。
  • 1965年， 鲁滨逊提出了归结原理，实现了自动定理证明的重大突破。
  • 1969年成立了国际人工智能联合会议，是人工智能发展史上的一个重要里程碑，它标志着人工智能这门新兴的学科已得到世界公认。

（3）发展时期（1970年以后）：

  • 1970年，创刊了人工智能杂志。
  • 1972年，法国科学家科默寥尔提出并实现了逻辑程序设计语言PROLOG。
  • 1972年，斯坦福大学的肖特里费研制用于诊断和治疗感染疾病的专家系统MYCIN。
  • 80年代，人工智能以推理技术、知识获取、自然语言理解和机器视觉为主，开始了不确定性推理、非单调推理、定性推理方法的研究。专家系统和知识工程在全世界得到了迅速的发展。
  • 90年代，专家系统，机器翻译、机器视觉、问题求解等方面的研究已有实际应用。

（4）重要事件：

  • 1977年，费根鲍姆在第5届国际人工智能会议上提出了“知识工程”，对以知识为基础的智能系统的研究与建造起到了重要作用。
  • 日本在1981年宣布了第五代计算机的发展计划，在1991年第12届国际人工智能联合会议上展出了PSI－3智能工作站和4×4PSI－3构成的模型机系统。

第二章

知识的定义和特点：

①　知识是经过消减、塑造、解释、选择和转换的信息。（费根鲍姆 Feigenbaum）

②　知识是由特定领域的描述、关系和过程组成的。（伯恩斯坦 Bernstein）

③　知识=事实+信念+启发式规则。（海叶斯-罗斯 Heyes-Roth）

(知识是人类对客观世界的认识和掌握，决定了知识具有以下特点）

（1）相对正确性

（2）不确定性

（3）可表达性

（4）可利用性

知识的分类：

根据知识表达的内容分类：

（1）事实性知识——知识的一般直接表示

（2）过程性知识——描述做某件事的过程，使人或计算机可以照此去做

（3）行为性知识——不直接给出事实本身，只给出它在某方面的行为

（4）实例性知识——只给出一些实例，关于事务的知识隐藏在这些事例中

（5）类比性知识——只给出它与其他事务的某些相似之处

（6）元知识——如何使用知识的知识

知识表示的定义和分类：

知识表示方法是研究用机器表示知识的可行性、有效性的一般方法，是一种数据结构与控制结构的统一体，既考虑知识的存储又考虑知识的使用。

知识表示可看成是一组事物的约定，以把人类知识表示成机器能处理的数据结构。对知识进行表示的过程就算把知识编码成某种数据结构的过程。

知识表示方法可分为：

  1. 陈述性知识表示——主要用来描述事实性知识。
  2. 过程性知识表示——主要用来描述规则性知识和控制结构知识。

一阶谓词表示法<重点>

  1. 定义谓词及个体，确定每个谓词及个体的确切含义。
  2. 根据所要表达的事物或概念，为每个谓词中的变元赋以特定的值。
  3. 根据所要表达的知识的语义，用适当的连接符将各个谓词连接起来形成谓词公式。

优点：（1）严密性（2）自然性（3）知识易表达（4）易于实现

缺点：（1）效率低（2）灵活性差（3）组合爆炸

*产生式表示法

<期中原题>产生式系统的组成：

  1. 规则库：用于描述某领域内知识的产生式集合。
  2. 数据库：又称为事实库，用来存放输入事实、外部数据库输入的事实以及中间结果和最后结果。
  3. 推理机：又称为控制系统，由一组程序组成，用来控制协调规则库和数据库的运行，包含了推理方式和控制策略。

<期中原题>产生式系统的三种推理方式：

  1. 正向推理（从已知事实出发，通过规则求得结论）
  2. 反向推理（从目标（作为假设）出发，反向使用规则，求得已知事实。）
  3. 双向推理（既自顶向下、又自底向上的推理方式）
  4. 混合推理（正向推理和逆向推理结合使用）

优点：（1）清晰性（2）模块化（3）自然性（4）可信度因子

缺点：（1）组合爆炸问题（2）控制饱和问题

一个框架通常由框架名、槽名、侧面和值这四部分组成。其一般结构如下：

​ … … … … … … … … …

​ … … … … … … … … …

槽名n：侧面名1 值1，值2，…，值r1

​ 侧面名2 值1，值2，…，值r2

​ … … … … … … … … …

​ 侧面名mn 值1，值2，…，值rmn

约束：约束条件1

​ 约束条件2

​ … … … … … … … … …

优点：（1）继承性（2）结构化（3）自然性（4）推理灵活多变

缺点：（1）不善于表达过程性知识，常和产生式结合使用

4.语义网络表示法<重点>

概念：

​ 语义网络是一种通过概念及其语义联系来表示知识的有向图，其中有向图的结点表示各种事物、概念、情况、属性、状态、事件和动作等，有向图的弧是有方向和标注的，方向表示结点之间的主次关系，不可随意调换。

结构：

​ 从结构上看，语义网络由最基本的语义单元组成。这些语义单元称为语义基元，用如下三元组表示
$$
（节点1，弧，节点2）
$$
​ 当把多个语义基元用相应的语义联系关联在一起时就形成了一个语义网络。

用语义网络表示知识的步骤

  1. 确定问题中的所有对象以及各对象的属性

  2. 分析并确定语义网络中所讨论对象间的关系

  3. 根据语义网络中所涉及的关系，对语义网络中的节点及弧进行整理，包括增加节点、弧和归并节点等。

  • 若是ISA/AKO/AMO等类属关系，下层可继承上层的属性，整理共同属性，抽取出共同属性，加入上层节点中避免数据冗余。
  • 若含有情况的表示，则设立情况节点，从该节点引出多条弧将各相关节点连接起来
  • 若含有动作关系，则设动作节点，分析动作的主体和客体
  • 若含有逻辑组成关系，即含“与”和“或”关系，可设立“与”节点和“或”节点
  • 若含有全称量词的复杂问题，则应采用亨得里克的网络分区技术
  • 若是规则性知识，则应分析问题中的条件和结果，并将它们作为语义网络中的两个节点，然后用If-then弧连接起来

  1. 分析语义网络中是否含有要表示的知识及所有对象

  2. 为各对象标示属性

优点：（1）结构性（2）联想性（3）自索引性（4）自然性

缺点：（1）非严格性（2）推理规则不十分明确

  • 继承性

  • 多态性：指一个名字可以有多种予以语义，多种解释。

第三章

推理的定义：

从已知事实出发，运用已掌握的知识，找出其中蕴含的事实，或归纳出新的事实，

这一过程通常称为推理，即推理就是按某种已知判断推出另一判断的思维过程。

确定性推理方式及分类：

确定性推理是指在推理时所用的知识都是精确的，推出的结论也是确定的，其真值为真，或者为假，没有第三种情况出现。

分类：

  1. 自然演绎推理
  2. 归结演绎推理

推理逻辑：

  1. 演绎推理
  2. 归纳推理
  3. 类比推理
  4. 默认推理

模式匹配：

模式匹配是指对两个知识模式的比较与耦合，即检查这两个知识是否完全一致或近似一致。

若两个知识完全一致或者虽不一致但其相似程度落在指定的限度内，则称它们是可以匹配的，否则不可匹配。

模式匹配可分为确定性匹配与不确定性匹配：

  1. 确定性匹配是指两个知识模式完全一致，或者经过变量代换后变得完全一致。

  2. 不确定性匹配是指两个知识模式不完全一致，但从整体上看，它们的相似程度落在规定的范围内。

<重点>代换：

目的：使得某些变元被另外的变元、常量或函数取代，使之不再在公式中出现。

代换的复合步骤：

设 $\theta={t_1/x_1,t_2/x_2,…t_n/x_n}$

​ $\lambda={u_1/y_1,u_2/y_2,…,u_n/y_n}$

是两个代换，则此两个代换得复合$\theta∘ \lambda$也是一个代换，它是从

${t_1\lambda/x_1,t_2\lambda/x_2,…t_n\lambda/x_n,u_1/y_1,u_2/y_2,…,u_n/y_n}$

中删去如下两种元素：

先删除：$u_i/y_i$ 当$y_i\in{x_1,x_2,…,x_n}$

后删除：$t_i\lambda/x_i$ 当$t_i\lambda=x_i$

<重点>合一：

最一般合一算法：

(1) 令$F={F_1,F_2},k=0,F_k=F,σ_k=ε (空代换)$

(2) 若$F_k$只含有一个表达式， 则算法停止

(3) 找出$F_k$的差异集$D_k$

(4) 若$D_k$中存在元素$x_k$和$t_k$，其中$x_k$是变元，$t_k$是项，且$x_k$不在$t_k$中出现，则

​ $σ_{k+1}=σ_k∘{t_k/x_k}$

​ $F_{k+1}=F_k{t_k/x_k}$

​ $k=k+1$

转（2）若不存在变元$x_k$和项$t_k$ ，则转（5）

(5) 算法终止，F的最一般合一不存在。

冲突消解策略：

  • 按就近原则排序
  • 按已知事实的新鲜性排序
  • 按匹配度排序
  • 按领域问题特点排序
  • 按上下文限制排序
  • 按条件个数排序
  • 按规则的次序排序

<重点>子句：

谓词公式化为子句集步骤：

  1. 消去->。

  2. ~ 紧靠谓词

  3. 相同约束变元换名

  4. 消去存在量词

  • 在全称量词前，用个体常量代替
  • 在全称量词后，用全称量词的函数代替

  1. 把全称量词移到最左边

  2. 把公式变为合取范式

  3. 消去全称量词

  4. 使不同句子中的变元不同名

  5. 消去合取词，生成子句集

<重点>鲁滨逊归结原理：

检查子句集S中是否包含空子句，若包含，则S不可满足；若不包含，就在子句集中选择合适的子句进行归结，一旦通过归结能推出空子句集，就说明子句集S是不可满足的。

证明问题

过程：

  1. 否定结论G，得到~G

  2. 将前提条件F和~G化为子句集S。

  3. 运用归结原理，反复对子句集S进行归结，归结出空子句，证明S不可满足。证明F->G为真。

求解问题

过程：

  1. 把已知条件用谓词公式表示出来，并且化为子句集S

  2. 把问题用谓词公式表达并否定，于answer构成析取式（变元与前一致）

  3. 把2转化为子句集并入S中，得到S’

  4. 对S’进行归结

  5. 等到answer。

<重点>归结策略：

删除策略

  1. 纯文字删除
  2. 重言式删除
  3. 包孕删除法

限制策略

  1. 支持集策略（完备的）

​ 每一次归结时，参加归结的子句至少有一个是由目标子句否定所得的子句或者是它的后裔。

  1. 线性输入策略（不完备的）

​ 要求参加归结的两个子句中必须至少有一个是初始子句集中的子句

  1. 单文字子句策略（不完备的）

​ 要求参加归结的两个子句中至少有一个是单文字子句（一个句子只包含一个文字，则为单文字子句）

  1. 祖先过滤形策略（完备的）

​ 要求参加归结的两个子句必须至少满足下列两个条件的任何一个。

  • C1和C2中至少有一个是初始子句集中的子句。
  • 如果两个子句集都不是初始子句集中的子句，则一个应是另一个的祖先。

不确定性推理的定义：

从不确定性的初始证据出发，通过运用不确定性的知识，最终推出具有一定程度的不确定性但却是合理或者近乎合理的结论的思维过程。

1.最大最小法

2.概率方法

3.有界方法

1．模型方法

1.1 数值方法

​ （1）主观Bayes方法

​ （2）可信度方法

​ （3）证据理论

​ 1.1.2 模糊推理

<重点>逆概率方法求解不确定推理问题：

Bayes定理：

设$B, A_1, A_2, … ,A_n$ 为一些事件，$P(B)>0, A_1, A_2, … , A_n$互不相交，$P(A_i)>0,i=1,2,…,n,$且$\sum_{i=1}^{n}P(A_i)=1,$对于$k=1,2,…,n$

有
$$ P(A_k|B)=\frac{P(A_k)P(B|A_k)}{\sum_{i=1}^{n}P(A_k)P(B|A_k)}
$$
