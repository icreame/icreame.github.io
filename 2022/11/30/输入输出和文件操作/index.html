输入输出和文件操作

Created2022-11-30Updated2022-12-09

输入输出

与输入输出流操作相关的类

  • istream是用于输入的流类，cin就是该类的对象

  • ostream是用于输入的流类，cout就是该类的对象

  • iostream是既能用于输入，又能用于输出的类

  • ifstream是用于从文件读取数据的类

  • ofstream是用于向文件写入数据的类

  • fstream是既能从文件读取数据，又能向文件写入数据的类

标准流对象

  • 输入流对象：cin（可被重定向为从文件中读取数据）
  • 输出流对象：cout（可被重定向为向文件中写入数据）
    • cerr：与标准错误输出设备相连（向屏幕输出出错信息）（不适用缓冲区，直接向显示器输出信息）
    • clog：与标准错误输出设备相连（向屏幕输出出错信息）（输出到clog中的信息先会被存放在缓冲区，缓冲区满或者刷新时才输出到屏幕上）

文件和流

文件建立

  • ofstream outfile(“clients.dat”,ios::out);
    • ofstream是fstream中定义的对象
    • outfile是我们定义的ofstream对象
    • cilents.dat是我们创建的文件名，可以是绝对路径，也可以是相对路径


#include<iostream>
#include<fstream>
#include<vector>
#include<algorithm>
using namespace std;
int main(){
    vector<int> v;
    ifstream srcFile("in.txt",ios::in);
    ofstream destFile("out.txt",ios::out);
    int x;
    while(srcFile>>x){
        v.push_back(x);
    }
    sort(v.begin(),v.end());
    for(int i=0;i<v.size();i++){
        destFile<<v[i]<<"";
    }
    destFile.close();
    srcFile.close();
    return 0;
}
